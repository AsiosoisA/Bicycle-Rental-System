## Use Case Description: 회원 탈퇴

| Actor Action                        | System Response                                   |
| ----------------------------------- | ------------------------------------------------- |
|                                     | 1. 회원 탈퇴 버튼을 화면에 표시한다.              |
| 2. 회원 탈퇴 버튼을 클릭한다.       |                                                   |
|                                     | 3. 탈퇴 확인 팝업 메시지를 표시한다.              |
| 4. 팝업 메시지의 “확인”을 클릭한다. |                                                   |
|                                     | 5. 회원 탈퇴 처리 후 탈퇴 완료 메시지를 출력한다. |
