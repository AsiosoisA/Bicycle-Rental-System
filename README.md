# SWE-18

### 공유 자전거 대여 시스템

공유 자전거 대여 시스템으로, 사용자는 회원 가입 후 자전거를 대여·반납하거나 예약할 수 있습니다.
대여소 검색, 대여 정보 조회, 식당 예약 연동 등의 기능이 포함되며, 관리자는 자전거 및 대여소 관리, 이용 통계 확인 등을 수행합니다.
